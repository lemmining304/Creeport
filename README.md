# Creeport – A linguagem de programação simples

## Como instalar

1. Vá no Termux e use:

```
pkg install git python
```

2. Clone o repositório:

```
git clone https://github.com/lemmining304/Creeport.git
```

3. Vá ao repositório:

```
cd Creeport
```

4. Execute o creeport.py:

```
python3 src/creeport.py examples/hello.cree
```

E pronto, você tem Creeport
